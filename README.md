# Game of Life
At this moment this realization has limited field, but in future I want to create endless realization (as a separate project)

### Requirements
* JDK 17 (or later)

### Run
```bash
$ git clone https://github.com/maxsavitsky/GameOfLife.git
$ cd GameOfLife
$ ./gradlew run
```
